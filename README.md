# NBA Fantasy HUB

NBA Fantasy HUB is a project designed to scrape, manipulate, and display the statistics over 500 NBA players, built for helping Fantasy Basketball players to create the best performing team possible, and with a coming machine learning model to predict the best fantasy performers of the upcoming season.

## Current Version

Currently, this web application consists of:

- scraping the web
- collecting data of the nba players box scores and seasonal averages
- storing the data
- user friendly interface

### Technologies

- Backend: Java/SpringBoot
- Scraping: Python/Pandas
- Frontend: React
